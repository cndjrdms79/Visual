2. UpdateData 활용 계산기 메뉴얼

1) Visual Studio 2019를 실행하여 [새 프로젝트 만들기]를 클릭합니다.
2) [MFC 앱]을 클릭합니다.
3) 프로젝트 이름을 지정한 후, 솔루션 및 프로젝트를 같은 디렉터리에 배치합니다.
4) 애플리케이션 종류를 [대화 상자 기반]으로 설정 해줍니다.
5) 대화 상자 제목을 지정 해줍니다.
6) 사용자 인터페이스 기능을 마침 버튼을 눌러줍니다.
7) 솔루션 탐색기에서 리소스 파일을 누르고 확장자가 rc인 파일을 누릅니다.
8) Dilog에서 IDD_(지정한이름)_Dialog를 클릭합니다.
9) 대화 상자 편집기에서 숫자를 입력할 [Edit Control]을 3개 입력 하고
10) [+] 버튼을 입력할 [Button]을 1개 입력합니다
11) [Button] 을 클릭하여 Caption을 [+] 로 변경합니다.
12) 샘플 입력란을 우클릭하여 변수 추가를 누릅니다.
13) 범주를 컨트롤에서 값으로 이름을 m_a로 변수 형식을 int로 변경합니다.
14) 두번째 샘플 입력란을 우클릭하여 변수 추가를 누릅니다.
15) 범주를 컨트롤에서 값으로 이름을 m_b로 변수 형식을 int로 변경합니다.
16) 세번째 샘플 입력란을 우클릭하여 변수 추가를 누릅니다.
17) 범주를 컨트롤에서 값으로 이름을 m_c로 변수 형식을 int로 변경합니다.
11) [+] 버튼을 더블클릭하여 

	UpdateData(true);
	m_c = m_a + m_b;
	UpdateData(false);  

12) F5를 눌러 실행시킵니다.
